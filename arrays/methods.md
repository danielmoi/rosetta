# Arrays

## Length

JS
```
const arr = [1, 2, 3];
const length = arr.length
```


Swift
```
let arr = [1, 2, 3]
const length = arr.count
```


## Add
JS
```
const arr = [1, 2, 3]
arr.push(4)

// or
const newArr = arr.concat(4)
```

Swift
```
// need to use var to mutate
var arr = [1, 2, 3]
arr.append(4)
```
