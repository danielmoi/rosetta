# Dictionaries

JavaScript
```
const store = {
  lollies: 10,
  soda: 5,
};


store.chips = 5;
delete store.lollies;

const sodaCount = store.soda;
```


Swift
```
let store = [
  "lollies": 10,
  "soda": 5,
]

store["chips"] = 5;
store.remove(forKey: "lollies")

let sodaCount = store["soda"]

```
